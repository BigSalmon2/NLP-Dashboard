Guidelines:

  1. Use __ to represent the blank or mask.
  2. Use only one blank space between words.

Example:

  1. Users can __ current version of pywidgets with pip or conda. Installing ipywidgets with __ will also enable the extension for you.
  2. Users can 'install' current version of pywidgets with pip or conda. Installing ipywidgets with 'pip' will also enable the extension for you.
